General

The general tab contains various settings regarding AdsManager.

Email Admin
The admin notifications will be sent to that address.

Name Admin
The name of the admin.

Automatically Published
If set to No, the administrator will need to validate each ads before their publications.

Cron Type
How background tasks like ad expiration should be executed.

Activate Ad Preview before publication
Show the preview button in the ad edit page.

Validation on update
The administrator will need to re-validate the ads after their modifications.

Publication date
Will allow the user to choose the publication date of the ad.

Submission Type
You can choose which types of users can post ads.

Allow ads at root level
Forbid to post ads in parents categories.

Number of Categories
Number of categories in which an ad can be posted.

Number of Ads by User
Maximum number of published ads by users.

Metadata
You can select how to display the metadatas.

Display RSS Link
Will display the RSS link.

Display print button
Will display the print button.

Show accept rules checkbox
Will display the accept rules checkbox.

Redirect after saving
You can choose on which page the user will be redirected after creating the ad.

Field for global filter Module
Name of the field to use for global filtering.
The global filtering feature will use the searched keyword on all the pages of AdsManager.

Global filtering on user ads view
Apply global filtering on user ads view.
